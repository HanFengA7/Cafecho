# Introduction

## what is Cafecho ?

Cafecho is a blog written in golang.

## Technology stack

#### Front end : Vue.js

#### Back end : Golang + Gin + Mysql + GORM

## Operating environment

#### OperatingSystem: `Linux` or `Windows` or `Macos`

#### Golang: `1.20`

#### Mysql: `5.7`

## Install

### 1. clone project

    git clone git@github.com:HanFengA7/Cafecho.git

### 2. Installation dependency

    go mod tidy
